# ⚡ System-X-Override: Futuristic Cyberpunk Terminal HUD

System-X-Override waa interface (UI) casri ah oo ku dhex shaqeeya biraawsarka (Browser-based), looguna talagalay qaabka Hacker Terminal ama Cyberpunk Dashboard. Mashruucan wuxuu isku dhex dhabbaa nashqadaynta casriga ah ee **Tailwind CSS** iyo dareenkii hore ee mashiinnada retro-ka (CRT Monitors).

---

## ✨ Awoodaha UI-gaan (Features)

* 📺 **CRT Scanlines & Flicker Effect:** Saamaynta khadadka shaashadaha hore iyo gilgashada iftiinka ee caanka ku ah terminal-ada rasmiga ah.
* 🟩 **Matrix Aesthetic:** Midabka cagaarka ee neon-ka ah (`#00ff41`) oo indhaha soo jiidanaya, kana dhigaya naqshadda mid sci-fi ah.
* 📊 **Multi-Panel Layout:** Qaybo u kala jaysan siday u shaqeeyaan sida *Packet Sniffer [ETH0]*, *Network Topology*, *Active Alerts*, iyo *Decryption Matrix*.
* 📱 **Fully Responsive Layout:** Wuxuu isticmaalaa Tailwind Grid System si uu ugu fadhiisto shaashad kasta si sax ah.

---

## 🛠️ Tiknoolajiyada la isticmaalay (Tech Stack)

* **HTML5** - Qaabdhismeedka iyo laf-dhabarta bogga.
* **Tailwind CSS** - Naqshadaynta, dynamic styling, iyo habaynta Layout-ka (Grid/Flexbox).
* **JavaScript (`java.js`)** - Maskaxda maamusha dhaqdhaqaaqa iyo shaqada terminal-ka.
* **Custom CSS (Variables & Animations)** - Loogu talagalay saamaynta scanlines-ka iyo animation-ka baararka (CPU/RAM).
* **Google Fonts** - Font-ka mashiinnada ee `JetBrains Mono`.

---

## 🧠 Doorka Udub-Dhexaadka ah ee JavaScript (`java.js`)

JavaScript-ku kaliya maaha koodh caawiye ah, waa mishiinka rasmiga ah ee maamula shaqada iyo firfircoonida dashboard-ka, wuxuuna mas'uul ka yahay:
* 🕒 **Live Uptime Counter:** Saacad dynamic ah oo xisaabinaysa ilbiriqsi kasta oo nidaamku furan yahay.
* ⚡ **Interactive Cyber Actions:** Wuxuu nidaamiyaa marka la gujiyo badanka weyn ee `EXECUTE BYPASS` si nidaamka loo dhex galo ama fariimo loogu soo bandhigo isticmaalaha.
* 🛰️ **Data Simulation:** Wuxuu si toos ah u soo saaraa koodhadhka qarsoon (*Decryption Matrix*) iyo xogta shabakadda (*Packet Sniffer*) si loo siiyo muuqaal dhab ah oo shaqaynaya.
